#PING目标机器数据

本节点将返回该节点的正向PING数据，默认不传参的情况下将显示两小时内的数据，可以传入开始时间与结束时间显示响应数据

##接口方法
GET

##接口地址
/api/ping.json

##接口入参
| **参数** | **格式** | **示例** | 
| :--- | :--- | :--- |
|ip|ip|127.0.0.1|
|starttime|yyyy-mm-dd hh:ii|2017-09-14 14:44|
|endtime|yyyy-mm-dd hh:ii|2017-09-14 14:44|
