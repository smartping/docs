# Mapping

本节点将返回全国延迟Ping数据，不加参数默认显示前一分钟的数据

## 接口方法

GET

## 接口地址

/api/alert.json

## 接口入参

| **参数** | **格式** | **示例** |
| :--- | :--- | :--- |
| d | yyyy-mm-dd hh:mm | 2017-09-16 10:20 |
